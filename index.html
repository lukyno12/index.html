Programátorský maratón

V sobotu som sa naplno pustil do tvorby vlastného „Discord bota" — zložitého programu na automatickú správu komunikácie v internetovej skupine.

Bol to skutočný maratón. Takmer celý deň som písaním a ladením kódu tvoril celú logiku programu a ošetroval technické chyby.

"Keď ti kód prvýkrát zafunguje bez erroru o 23:47 — to je iný pocit."

Večer som si od monitora odbehol na slávnostnú vigíliu, čo mi pomohlo oddýchnuť.

discord-bot.js
// hlavný handler pre príkazy
client.on('messageCreate', async msg => {
  if (!msg.content.startsWith('!')) return;
  const [cmd, ...args] = msg.content.slice(1).split(' ');
  await handleCommand(cmd, args, msg);
});

// auto-moderácia a správa servera
async function handleCommand(cmd, args, msg) {
  const cmds = { ban, kick, mute, warn, stats };
  if (cmds[cmd]) await cmds[cmd](args, msg);
}
